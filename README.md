# Gestión de Contactos - Programación de Interfaces Gráficas

**Universidad Politécnica Salesiana**  
**Unidad 4: Gestión de Dependencias**  

Proyecto de aplicación de escritorio para gestión de contactos, desarrollada en Java con Swing, migrada a Maven, con mejoras en la interfaz mediante FlatLaf y manejo de archivos JSON usando Gson. Incluye exportación/importación de contactos en CSV y JSON.

---

## 📋 Características principales

- ✨ Interfaz moderna con **FlatLaf** (tema claro).
- 📁 Exportar contactos a **CSV** (con barra de progreso).
- 📂 Importar/exportar contactos a **JSON** (usando Gson).
- 🖱️ Menú contextual en la tabla con todas las opciones.
- ⌨️ Atajo de teclado `CTRL + E` para exportar a CSV.
- 📊 Pestañas: Lista de Contactos y Estadísticas (placeholder).
